**# Tripistry**

Tripistry is a COS221 project that provides a web-based platform where travel agencies can publish vacation and travel packages, and travelers can browse and book trips directly through the system. The platform supports both individual and group bookings, making trip planning more flexible and collaborative.

**Features**
- Agency registration and management
- Create and manage travel packages
- Browse available trips and vacation deals
- Individual trip booking
- Group booking functionality
- User authentication and account management
- Booking management dashboard
- Responsive web interface

**Project Goals**

The goal of Tripistry is to simplify the interaction between travel agencies and travelers by providing a centralized platform for discovering, managing, and booking travel experiences.
